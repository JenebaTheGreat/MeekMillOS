# Empire Command Center
### Robert Williams — Business & Operations Dashboard

A private AI-powered command center for tracking and scaling music, business operations, and investment portfolio. Built by **AI Atelier**.

---

## What's Inside

| Tab | Description |
|-----|-------------|
| **Overview** | Net worth, independence milestone, revenue mix, business timeline |
| **Revenue** | Stream estimates, monthly tracker, Before/After independence chart |
| **Catalog** | Full RIAA certification table — albums, solos, featured |
| **Operations** | All business entities, contractual history, AI ops log |
| **Opportunities** | White space + blue ocean analysis with revenue potential |
| **Weekly Brief** | AI-generated weekly recommendations based on new data input |

---

## Deploy to Vercel

### Option 1 — Vercel CLI
```bash
npm i -g vercel
vercel
```

### Option 2 — GitHub Import
1. Push this repo to GitHub
2. Go to [vercel.com/new](https://vercel.com/new)
3. Import the repository
4. Framework: **Other**
5. Click **Deploy**

No build step required. Pure static HTML.

---

## Add New Monthly Data

Navigate to the **Revenue** tab and use the Monthly Revenue Tracker form to input new data by month. The stacked bar chart updates live. Then go to **Weekly Brief** to generate a fresh AI recommendation brief.
